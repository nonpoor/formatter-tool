# PROJECT_BRIEF

## 项目定位
`formatter-tool` 是一个面向中文学习/作业场景的文本整理工具。  
核心价值：把 AI/网页复制内容转成更稳定的 Word/WPS 可用结构，减少手工排版时间。

## 当前核心目标
1. 粘贴内容后结构稳定（标题、段落、列表、表格、引用）。
2. 导出 `.docx` 可直接交付，复制到 Word/WPS 时尽量不乱。
3. 学术模板场景下，输出层级可用于目录生成（Heading 1-3）。

## 目标用户
1. 学生（课程论文、实验报告、作业整理）。
2. 需要把 AI 结果快速整理成文档的人。

## 非目标（当前阶段）
1. 不做完整论文写作 Agent。
2. 不做模板上传与复杂模板映射。
3. 不做多人协作后台系统。

## 成功标准（MVP）
1. 常见粘贴源（Markdown/HTML/混合）可被稳定解析。
2. 列表与层级在预览、复制、导出三端一致。
3. 学术模板中，Word 可基于标题样式生成目录。
