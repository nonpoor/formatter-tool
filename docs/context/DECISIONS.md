# DECISIONS

## 2026-04-03：粘贴策略采用“结构优先”兜底
- 决策：默认仍优先 `text/plain`，但当 `text/html` 结构明显更完整时，采用 HTML 投影后的纯文本。
- 原因：解决“整段复制正常、单段复制丢层级”的高频问题。
- 影响：`src/features/formatter/paste.ts`。

## 2026-04-03：列表识别增强
- 决策：增强前缀识别（圆点/全角数字/中文编号），并支持“说明行 + 列表行”拆分。
- 原因：提升中文作业内容粘贴稳定性。
- 影响：`normalize`、相关测试集。

## 2026-04-03：UI 采用当前苹果风格版本
- 决策：保持当前 UI 方案，不在此阶段继续视觉大改。
- 原因：把重心放到结构与导出稳定性。

## 2026-04-03：学术结构化重构方向确定
- 决策：
  1. 学术模板启用 `academic` 模式，默认模板维持 `legacy`。
  2. 目标是“目录友好”，不自动插 TOC 字段。
  3. 规则作用于预览/复制/docx 三端一致。
- 状态：进行中（见 `HANDOFF.md`）。
