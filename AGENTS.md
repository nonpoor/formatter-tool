# AGENTS

## Project-specific mandatory rules
1. 先读代码再写文档
- 任何文档修改前，必须先阅读对应代码与现有文档。

2. 不要编造项目目标
- 目标与范围以 `docs/context/PROJECT_BRIEF.md` 为准。
- 不确定信息必须标注 `Need Verification`，禁止猜测填充。

3. 任务结束自动交接（强制）
- 每次任务结束都必须更新：
  - `docs/context/HANDOFF.md`
  - `docs/context/TASKS.md`
- 若出现新技术决策，额外更新 `docs/context/DECISIONS.md`。

4. 文档与流程导航
- 新线程入口：`docs/context/START_HERE.md`
- 交接流程：`docs/context/WORKFLOW.md`
